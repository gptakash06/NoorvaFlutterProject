# Noorva Flutter Project

A Flutter implementation of Noorva UI task based on provided Figma design.

---

## ✨ Features

- Responsive UI
- GetX State Management
- Reusable Widgets
- Hero Animation
- Suggestion Slider
- Bottom Chat Input UI
- Google Sans Font Integration
